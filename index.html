Slots

  • Anastomoses - Indicates whether the vessel anastomoses with another vessel.
  • Biomarkers - A biomarker associated with the vessel.
  • BiomarkersID - The ID of the biomarker in HUGO Gene Nomenclature Committee (HGNC)
  • BiomarkersLabel - The preferred name of the biomarker in HUGO Gene Nomenclature Committee (HGNC)
  • BodyPart - A mapping from vessel to organ or part of the body.
  • BodyPartUberon - The UBERON ID of the BodyPart.
  • BranchSequence - The order in which vessels branch off of the BranchesFrom vessel. Vessels can have the same value if they branch off the BranchesFrom vessel at the same place. A value of 999 means the branching sequence will be added in a future version of this table.
  • BranchesFrom - The 'parent' vessel that is one step closer to the heart. For veins it is 'drains to' rather than branches from.
  • CellType - A type of cell associated with the vessel.
  • CellTypeID - The cell type ID in Cell Ontology.
  • CellTypeLabel - The preferred name of the cell in Cell Ontology.
  • CoordX - The X coordinate for the approximate location of selected vessels. This is more to show relative position (left vs right) rather than the exact location.
  • CoordY - The Y coordinate for the approximate location of selected vessels. This is more to show relative position (above vs below) rather than the exact location.
  • FMA - The ID of the vessel in the FMA ontology.
  • FMALabel - The preferred name of the vessel in FMA.
  • ForBranchesSee - For some vessels, like the left and right renal arteries, rather than showing all the branches of both arteries, a "virtual" merged vessel is created (e.g., just "renal artery"). The branches are only added once to that virtual vessel. This field indicates the name of the virtual vessel that has the branches.
  • MergedVessel - This field contains a "1" if it is virtual merged vessel used to show the branches of other vessels.
  • PathFromHeart - The list of branches leading from a heart chamber to the vessel.
  • Reference - A reference for the website. For example, if the ReferenceURL is a journal article, then this field is the reference (authors, title, journal, etc.) of the article.
  • ReferenceDOI - The DOI of the reference if applicable.
  • ReferenceURL - The website describing the vessel and where it branches from.
  • Sex - Indicates whether the vessel is only found in males or females.
  • UBERON - The ID of the vessel in the UBERON ontology.
  • UBERONLabel - The preferred name of the vessel in UBERON.
  • Vessel - The name of the blood vessel. The unique item (primary key) in this table.
  • VesselType - Either heart chamber, artery, arteriole, capillary, venule, vein, sinusoid, or sinus.
